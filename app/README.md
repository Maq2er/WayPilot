# Санья — React PWA

Основная мобильная TypeScript/PWA-версия офлайн-путеводителя. Данные мест
берутся напрямую из `../sanya_locations_v3.geojson`.

## Локальный запуск

```powershell
pnpm install
pnpm dev
```

## Production-сборка

```powershell
pnpm build
pnpm preview
```

Готовые файлы находятся в `dist/`.

## Установка на iPhone

1. Опубликовать содержимое `dist/` на HTTPS-хостинге, например GitHub Pages.
2. Открыть адрес в Safari и дождаться полной загрузки.
3. Нажать «Поделиться» → «На экран Домой» → «Добавить».
4. Запустить приложение с домашнего экрана.
5. Включить авиарежим и проверить повторный запуск.

Service Worker кэширует интерфейс и GeoJSON. Внешние Apple Maps и проверка
актуальных часов заведений требуют интернет.

Локальные записи разговорника размещаются в `public/audio/phrases/`. После
сборки MP3/WAV включаются в PWA-кэш. Если файла нет, приложение пробует
установленный китайский системный голос и показывает «Аудио недоступно», если
голоса тоже нет.

Файл `../sanya_offline_guide_v3.html` остаётся независимым резервным вариантом,
который можно хранить в приложении «Файлы».
