# Локальные аудиофайлы

Актуальные заглушки и будущие записи перенесены в:

`public/assets/audio/phrases/<category>/`.

Пути уже прописаны в `src/data/phrases.ts`. Нужно заменить соответствующий
`.mp3`, не меняя имя файла. После следующей сборки приложение автоматически
добавит запись в PWA-кэш.

Web Speech API используется только как необязательный fallback, если на
устройстве установлен китайский системный голос. Для критичных сценариев
следует использовать проверенные локальные MP3.
