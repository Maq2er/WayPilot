# Audio phrase placeholders

Файлы в подпапках — намеренно невалидные текстовые MP3-заглушки. Они нужны,
чтобы заранее зафиксировать структуру и имена ассетов.

Замените каждый файл настоящей MP3-дорожкой, сохранив имя. Рекомендуемые
параметры: mono, 44.1 kHz, 96–128 kbps, без музыки и длинной тишины.

До замены приложение безопасно покажет «Аудио недоступно» или попробует
китайский системный голос устройства.
